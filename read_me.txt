R code for implementing the clinical trial recruitment prediction method from "Interim recruitment prediction for multi-centre clinical trials"

model_functions.R : functions needed to construct the models
data_sim.R : simulate dataset
MLE.R : fit models using maximum likelihood estimation and predict from best model
BMA.R : fit models by obtaining Bayesian posteriors and average predictions across the models 

Delay process: folder containing code which allows for delayed centre initiations
